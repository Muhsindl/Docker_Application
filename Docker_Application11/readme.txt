docker pull muhsindolu/docker-compose:0.2	-> Docker Compose dosyasını docker hub'dan çeker
docker-compose up -d				-> Docker Compose yapısını çalıştırabilirsiniz (Compose bulunduğu dosya dizinine gidip)
http://localhost:8080               -> Çalıştığını göreceksiniz
http://localhost:8080/user			-> POSTMAN ile json formatında "{ "name": "Muhsin", "age": 23 }" şeklinde veri ekleyebilirsiniz.
